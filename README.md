# 2025 Fall Introduction to Formal Mathematics with Lean 4

This repository contains the materials for the hobby course "Introduction to Formal Mathematics with Lean 4", offered in Fall 2025 at Beijing Institute of Technology.

- [course repository](https://github.com/sun123zxy/2025fall-lean4-teach)
- [online documentation](https://sun123zxy.github.io/2025fall-lean4-teach/)

## Installation

First make sure you have installed Git, VSCode and Lean 4 extension for VSCode correctly. Then execute the following commands in your terminal:

```bash
git clone git@github.com:sun123zxy/2025fall-lean4-teach.git
cd 2025fall-lean4-teach
lake exe cache get
```
